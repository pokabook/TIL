# Edge Location

### Edge Location

---

- Amazon CloudFront / Route53을 위한 캐시 서버(Cache Server)들의 모음
- 엣지 로케이션은 AWS 전세계 리전에서 빠른 접근성을 위한 글로벌 네트워크 인프라 (400개 지점)
- Lambda@Edge (가까운 지역 위치에서 서버리스 코드 실행) 를 통해 성능을 개선하고 지연 시간 단축
