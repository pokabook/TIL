# Kubernetes 구축 환경

### Kubernetes 구축 환경

- **로컬 환경**
    - 물리머신 내에서 구축
    - 대부분 테스트 혹은 스터디용
    - Minikube, Docker Desktop for Mac/Windows
- **Manual 구축 with 도구**
    - 도구를 사용해서 원하는 곳에 클러스터를 구축
    - kubeadm, kops, Rancher 등등
- **관리형 Kubernetes 서비스 (CSP 제공)**
    - 퍼블릭 클라우드에서 제공하는 관리형 서비스를 사용
    - AWS EKS, GCP GKE, Azure AKS
